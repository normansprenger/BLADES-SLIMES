Class: BackgroundObject

BackgroundObject(imagePath, x, y)

Represents an background object in the game, extending the `MovableObject` class.

Constructor

new BackgroundObject(imagePath, x, y)

Creates an instance of the class with an image and sets its position.
Parameters:
Name Type Description
imagePath string The path to the image to be loaded for the instance.
x number The x-coordinate for the position of the entity.
y number The y-coordinate for the position of the entity, which is adjusted based on height.
