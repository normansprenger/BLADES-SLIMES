Class: Level

Level(enemies, clouds, coins, backgroundObjects)

Represents a game level with various elements such as enemies, clouds, coins, and background objects.

Constructor

new Level(enemies, clouds, coins, backgroundObjects)

Creates an instance of the Level class. Initializes the level with the specified enemies, clouds, coins, and background objects. The `levelEndX` property is set to a default value of 4360, which represents the end point of the level.
Parameters:
Name Type Description
enemies Array.<Enemy> The array of enemy objects in the level.
clouds Array.<Cloud> The array of cloud objects in the level.
coins Array.<Coin> The array of coin objects in the level.
backgroundObjects Array.<BackgroundObject> The array of background objects in the level.
